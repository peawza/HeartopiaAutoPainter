# 🎲 คู่มืออัปเกรด Randomness (ไม่ทำให้รูปเพี้ยน)

## 📋 สรุปการเปลี่ยนแปลง

เพิ่ม **Randomness** ให้เหมือนมนุษย์มากขึ้น แต่**ไม่กระทบความแม่นยำของรูป**

---

## ✅ สิ่งที่เปลี่ยนแปลง

### 1. **Timing Randomness (เวลา)** - เพิ่มมากขึ้น ✨

| การตั้งค่า | เดิม | ใหม่ | เหตุผล |
|-----------|------|------|--------|
| **delay_min_ms** | 240ms | **200ms** | เร็วขึ้น เมื่อรีบ |
| **delay_max_ms** | 320ms | **400ms** | ช้าขึ้น เมื่อระมัดระวัง |
| **Range** | 80ms | **200ms** | หลากหลายมากขึ้น 2.5 เท่า! |

**ผลลัพธ์:**
```
เดิม: 240-320ms (แคบ)
ใหม่: 200-400ms (กว้างมาก)

ตัวอย่าง:
Action 1: 205ms (เร็วมาก)
Action 2: 387ms (ช้ามาก)
Action 3: 298ms (ปานกลาง)

= ความแปรผันสูง → เหมือนคนจริง!
```

**✅ ไม่กระทบรูป:** เพราะเป็นแค่ความเร็ว ไม่ใช่ตำแหน่ง

---

### 2. **Position Randomness (ตำแหน่ง)** - ลดลง! 🎯

| การตั้งค่า | เดิม | ใหม่ | เหตุผล |
|-----------|------|------|--------|
| **click_randomness_px** | ±25px | **±3px** | เพื่อความแม่นยำ |

**ผลลัพธ์:**
```
เดิม: ±25px (เบี่ยงเยอะ อาจทำให้รูปเพี้ยน)
ใหม่: ±3px (เบี่ยงนิดหน่อย แม่นยำมาก)

เป้าหมาย: (500, 300)
ตัวอย่าง:
- คลิก 1: (502, 298) ← เบี่ยง +2, -2
- คลิก 2: (499, 301) ← เบี่ยง -1, +1
- คลิก 3: (500, 303) ← เบี่ยง 0, +3

= ยังดูเหมือนคน แต่แม่นยำมาก!
```

**✅ ปลอดภัย:** ±3px เล็กมาก ไม่กระทบรูปเลย

**💡 เหตุผล:**
- Canvas แต่ละ cell อาจเล็กถึง 10-30px
- ±25px = เบี่ยงไปคนละ cell (รูปเพี้ยน!)
- ±3px = ยังอยู่ใน cell เดียวกัน (รูปไม่เพี้ยน!)

---

### 3. **Micro-Pause (หยุดคิด)** - เพิ่มขึ้น 🤔

| การตั้งค่า | เดิม | ใหม่ | เหตุผล |
|-----------|------|------|--------|
| **Probability** | 20% | **25%** | หยุดบ่อยขึ้น |
| **Duration Min** | 1.0s | **0.8s** | เร็วขึ้น (บางครั้ง) |
| **Duration Max** | 2.5s | **3.0s** | ช้าขึ้น (บางครั้ง) |

**ผลลัพธ์:**
```
เดิม: 1 ใน 5 ครั้ง หยุด 1.0-2.5s
ใหม่: 1 ใน 4 ครั้ง หยุด 0.8-3.0s (หลากหลายมากขึ้น)

ตัวอย่าง:
คลิก 1: ⚡ ไม่หยุด
คลิก 2: ⚡ ไม่หยุด
คลิก 3: 🤔 หยุด 0.9s (คิดเร็ว)
คลิก 4: ⚡ ไม่หยุด
คลิก 5: 🤔 หยุด 2.7s (คิดนาน)
```

**✅ ไม่กระทบรูป:** เป็นแค่การหยุด ไม่ใช่คลิกผิดที่

---

### 4. **Mistake Simulation (ข้อผิดพลาด)** - เพิ่มขึ้น ⚠️

| การตั้งค่า | เดิม | ใหม่ | เหตุผล |
|-----------|------|------|--------|
| **Probability** | 5% | **8%** | ทำผิดบ่อยขึ้น (เหมือนคน) |

**ผลลัพธ์:**
```
เดิม: 1 ใน 20 ครั้ง ทำผิด
ใหม่: 1 ใน 12 ครั้ง ทำผิด

ตัวอย่าง:
Action 1-11: คลิกถูก ✅✅✅✅✅✅✅✅✅✅✅
Action 12:   คลิกผิด (523, 287) ❌ → แก้ไข (500, 300) ✅
```

**⚠️ อาจกระทบรูปเล็กน้อย:**
- แต่ระบบจะแก้ไขทันที (คลิก 2 ครั้ง)
- ความผิดพลาดเล็กน้อยทำให้ดูเป็นธรรมชาติ

**💡 ทางเลือก:** ถ้าไม่ต้องการ สามารถตั้ง `"enable_mistakes": false`

### 5. **Speed Variation (ความเร็วแปรผัน)** - เพิ่มขึ้น 🚀

| การตั้งค่า | เดิม | ใหม่ | เหตุผล |
|-----------|------|------|--------|
| **Min** | 75% | **70%** | เร็วขึ้น (เมื่อรีบ) |
| **Max** | 125% | **135%** | ช้าขึ้น (เมื่อระมัดระวัง) |
| **Range** | 50% | **65%** | หลากหลายมากขึ้น |

**ผลลัพธ์:**
```
ปกติ: 0.250s

เดิม: 75-125%
- เร็ว:   0.188s (75%)
- ช้า:    0.313s (125%)

ใหม่: 70-135%
- เร็วมาก: 0.175s (70%)
- ช้ามาก:  0.338s (135%)

= หลากหลายมากขึ้น 30%!
```

**✅ ไม่กระทบรูป:** เป็นแค่ความเร็ว ไม่ใช่ตำแหน่ง

### 6. **Fatigue (ความเหนื่อยล้า)** - เพิ่มขึ้น 😴

| การตั้งค่า | เดิม | ใหม่ | เหตุผล |
|-----------|------|------|--------|
| **Slowdown/100** | 0.8% | **1.0%** | ช้าลงเร็วขึ้น |
| **Max Multiplier** | 1.5× | **1.6×** | ช้าสุดมากขึ้น |

**ผลลัพธ์:**
```
เดิม: ช้าลง 0.8% ทุก 100 actions (max 50% ช้ากว่าปกติ)
ใหม่: ช้าลง 1.0% ทุก 100 actions (max 60% ช้ากว่าปกติ)

Action 1-100:   0.250s (ยังไม่เหนื่อย)
Action 101-200: 0.253s (+1.0%)
Action 201-300: 0.255s (+2.0%)
...
Action 6000+:   0.400s (+60% = max)

= เหนื่อยเร็วขึ้น → เหมือนคนจริง!
```

**✅ ไม่กระทบรูป:** เป็นแค่ความเร็ว ยิ่งวาดนานยิ่งช้า

## 💡 คำแนะนำเพิ่มเติม

### ถ้าต้องการความแม่นยำสูงสุด (รูปซับซ้อน)
```json
{
  "click_randomness_px": 2,          ← ลดเป็น ±2px
  "mistake_probability": 0.05,       ← ลดกลับเป็น 5%
  "enable_mistakes": false           ← ปิด mistakes เลย
}
```

### ถ้าต้องการ Randomness สูงสุด (รูปง่าย)
```json
{
  "click_randomness_px": 5,          ← เพิ่มเป็น ±5px
  "mistake_probability": 0.10,       ← เพิ่มเป็น 10%
  "delay_min_ms": 150,               ← เร็วสุด 150ms
  "delay_max_ms": 500                ← ช้าสุด 500ms
}
```

### ค่าแนะนำ (ปัจจุบัน) - สมดุลดีที่สุด ⭐
```json
{
  "click_randomness_px": 3,          ← แม่นยำ
  "mistake_probability": 0.08,       ← ข้อผิดพลาดน้อย
  "delay_min_ms": 200,               ← Timing หลากหลาย
  "delay_max_ms": 400,
  "micro_pause_probability": 0.25,   ← หยุดคิดบ่อย
  "speed_variation_min": 0.70,       ← ความเร็วหลากหลาย
  "speed_variation_max": 1.35
}
```
