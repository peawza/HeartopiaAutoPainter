# ขั้นตอนการติดตั้งและรัน (Windows)
1 ติดตั้ง python 3+ ก่อน
https://www.python.org/ftp/python/3.14.3/python-3.14.3-amd64.exe

2 แตกไฟล์ Heartopia.Help-painter-main.zip ไว้ที่ desktop

3 ใช้ PowerShell (Run as administrator) ในการรัน
Start Menu → พิมพ์ PowerShell → Run as Administrator

4 ปลดล็อกสิทธิ์การรัน Script (ทำครั้งเดียว)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

5 เข้าไปยังโฟลเดอร์โปรเจกต์ แฟ้มต้องชื่อ Heartopia.Help-painter-main และต้องอยู่ที่ Desktop
cd "$HOME\Desktop\Heartopia.Help-painter-main"

6 สร้าง Virtual Environment ():
python -m venv .venv

7 ทำการ Activate
.\.venv\Scripts\Activate.ps1

8 ติดตั้ง Library ที่จำเป็น (สำคัญมาก)
pip install -r requirements.txt --upgrade

9 เริ่มใช้งานโปรแกรม
python main.py

10 ปิด Virtual Environment
เมื่อไม่ได้ใช้งานโปรแกรมแล้วให้รันปิด :
deactivate

🌟 ระบบใหม่! Enhanced Features - ป้องกันการตรวจจับ (Anti-Detection)

----------------------------------------------------------------------

เราได้เพิ่มระบบใหม่เข้ามาที่ช่วยให้การวาดภาพดูเหมือนมนุษย์จริงๆ มากขึ้น!

✅ ระบบ Delay อัจฉริยะ (Human-like Timing)
   - การกระจายเวลาแบบ Bell Curve (เหมือนคนจริง)
   - เคลื่อนเมาส์แบบเส้นโค้ง Bezier (ไม่ใช่เส้นตรง)
   - Position Jitter ±2 pixels (คนจริงไม่แม่นสมบูรณ์)
   - Micro-Pauses แบบสุ่ม 10% (เหมือนคนหยุดคิด)
   - เลือกได้ 3 โหมด: Fast (เร็ว), Default (ปกติ), Careful (ช้าแต่แม่น)

✅ Hardware Mouse Support (แนะนำ!)
   - ใช้ Arduino Leonardo/ESP32 เป็นเมาส์ USB จริง
   - ระบบมองเป็นเมาส์ของจริง (แทบไม่มีทางตรวจจับ!)
   - ความแม่นยำระดับ microsecond
   - หา COM port อัตโนมัติ (ไม่ต้องตั้งค่า)
   - ไม่มีก็ไม่เป็นไร จะใช้โหมด Software แทน

วิธีใช้งาน Enhanced Features

----------------------------------------------------------------------

วิธีที่ 1: ใช้ผ่าน GUI (แนะนำ - ง่ายสุด!)

1. เปิดโปรแกรม
   python main.py

2. เปิดใช้งาน Enhanced Timing
   ✅ ติ๊กถูก "เปิดใช้ Enhanced Timing"
   เลือก Delay Profile:
     - Fast = วาดเร็ว (0.15-0.25 วินาที/action)
     - Default = สมดุล (0.25-0.35 วินาที) ← แนะนำ!
     - Careful = ช้าแต่แม่น (0.40-0.60 วินาที)
   ✅ ติ๊กถูก "Position Jitter" (เลื่อน ±2px เหมือนคน)
   ✅ ติ๊กถูก "Micro Pauses" (หยุดคิด 10%)

3. เปิดใช้งาน Hardware Mouse (ถ้ามี Arduino)
   ✅ ติ๊กถูก "ใช้ Hardware Mouse ESP32/Arduino"
   ใส่พอร์ต (เช่น COM3) หรือปล่อยว่างให้หาอัตโนมัติ
   ถ้าไม่มี Arduino ก็ไม่เป็นไร จะใช้โหมด Software แทน

4. กดปุ่มวาด - เสร็จแล้ว!

----------------------------------------------------------------------

วิธีที่ 2: Setup Hardware Mouse (ไม่บังคับ แต่แนะนำ!)

----------------------------------------------------------------------

ขั้นตอนที่ 1: ซื้ออุปกรณ์

ซื้อ Arduino Leonardo ราคา ~500-1000 บาท
  - หาซื้อใน Shopee, Lazada, AliExpress
  - รุ่นที่ใช้ได้: Arduino Leonardo, Pro Micro, Arduino Micro
  - ทำไมต้อง Leonardo? เพราะมี USB HID support แบบ native!

ขั้นตอนที่ 2: อัพโหลด Firmware

1. ติดตั้ง Arduino IDE
   ดาวน์โหลด: https://www.arduino.cc/en/software

2. อัพโหลด Sketch
   - เปิด Arduino IDE
   - เปิดไฟล์: esp32\Arduino_Mouse\Arduino_Mouse.ino
   - เลือก Board: Tools → Board → Arduino Leonardo
   - เลือก Port: Tools → Port → (COM3, COM4, etc.)
   - กดปุ่ม Upload (ลูกศรชี้ขวา)
   - รอจน Upload เสร็จ (100%)

3. ทดสอบการเชื่อมต่อ
   python -m heartopia_painter.hardware_mouse

   ผลลัพธ์ที่ต้องการ:
   ✓ Auto-detecting Arduino...
   ✓ Found Arduino at: COM3
   ✓ Connected: Leonardo Mouse v1.1.0
   ✓ All tests passed!

4. หากเจอปัญหา
   - ลองเปลี่ยน USB port
   - กดปุ่ม Reset บน Arduino แล้วอัพโหลดใหม่
   - ดูคู่มือเพิ่มเติม: esp32\README_SETUP.txt

ขั้นตอนที่ 3: ใช้งานผ่าน GUI
   - เปิดโปรแกรม → ติ๊ก "ใช้ Hardware Mouse" → วาดได้เลย!

เปรียบเทียบ: ปกติ vs Enhanced

----------------------------------------------------------------------

ความปลอดภัย:        ปกติ (ปานกลาง) → Enhanced (ต่ำมาก)
การเคลื่อนที่:       เส้นตรง → เส้นโค้ง Bezier
Timing:             ตายตัว → สุ่มแบบ Bell Curve
ความแม่นยำ:         สมบูรณ์ → ±2px (เหมือนคน)
Micro-pauses:       ไม่มี → มีการหยุดคิด 10%
Anti-cheat:         จับได้ → แทบจับไม่ได้
