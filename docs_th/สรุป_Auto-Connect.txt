🔧 การตั้งค่า
───────────────────────────────────────────────────────────────

1. เปิด/ปิด Hardware Mouse
   → แก้ไข config.json:
   {
     "use_hardware_mouse": true    // เปิด
     "use_hardware_mouse": false   // ปิด
   }

2. ระบุพอร์ตเอง (ไม่ต้อง auto-detect)
   → แก้ไข mouse_config.json:
   {
     "arduino_port": "COM3"
   }

3. ตั้งค่าเพิ่มเติม
   → แก้ไข mouse_config.json:
   {
     "click_randomness_px": 25,
     "enable_fatigue": true,
     "enable_breaks": true,
     "session_time_limit_hours": 3.0
   }

───────────────────────────────────────────────────────────────
🐛 แก้ปัญหา
───────────────────────────────────────────────────────────────

❌ ปัญหา: Popup ไม่แสดง
✓ แก้ไข:
  1. เช็คว่า use_hardware_mouse = true
  2. เช็คว่ามี Arduino เสียบอยู่

❌ ปัญหา: หา Arduino ไม่เจอ
✓ แก้ไข:
  1. เปิด Device Manager (Windows)
  2. หา "Ports (COM & LPT)"
  3. หา "Arduino Leonardo" → จดพอร์ต (เช่น COM3)
  4. ใส่ใน mouse_config.json:
     { "arduino_port": "COM3" }

❌ ปัญหา: เชื่อมต่อล้มเหลว
✓ แก้ไข:
  1. ปิดโปรแกรมอื่นที่ใช้พอร์ต COM
  2. ถอดแล้วเสียบ Arduino ใหม่
  3. รีบูตคอมพิวเตอร์
  4. อัพโหลด firmware ใหม่:
     cd esp32 && upload.bat

💻 Technical Details
───────────────────────────────────────────────────────────────

Function: _test_hardware_mouse_connection()

Input:
  → None

Output:
  → dict {
      'success': bool,
      'enabled': bool,
      'port': str | None,
      'version': str | None,
      'message': str | None
    }

Flow:
  1. โหลด config.json
  2. เช็ค use_hardware_mouse
  3. ถ้าเปิด → หา Arduino
  4. เชื่อมต่อ
  5. ทดสอบ Ping
  6. ดึงข้อมูล Status
  7. ตัดการเชื่อมต่อ (จะเชื่อมใหม่ตอนวาด)
  8. คืนค่า result dict

Popup Timing:
  → แสดงหลังจาก MainWindow โหลดเสร็จ 500ms
  → ใช้ QTimer.singleShot()
